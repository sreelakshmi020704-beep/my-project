# 🏓 Pong Arena

A classic two-player Pong game built using **Python** and the **Turtle Graphics** library.

Challenge a friend in this retro arcade game where each player controls a paddle to keep the ball in play. Score points when your opponent misses the ball, and see who reaches the highest score!

## 🎮 Gameplay

* Two-player local multiplayer game.
* Players control paddles to bounce the ball back and forth.
* Each missed ball awards a point to the opposing player.
* The ball speeds up after every successful paddle hit, increasing the challenge.

## ✨ Features

* Two-player gameplay
* Smooth paddle movement
* Ball collision physics
* Dynamic ball speed
* Live scoreboard
* Automatic ball reset after each point
* Retro arcade experience

## 🎯 Controls

| Player       | Keys                     |
| ------------ | ------------------------ |
| Left Paddle  | **W** = Up, **S** = Down |
| Right Paddle | **↑** = Up, **↓** = Down |

## 🏆 Scoring

* If the right player misses the ball, the left player earns **1 point**.
* If the left player misses the ball, the right player earns **1 point**.
* The game continues until the players decide to stop.
