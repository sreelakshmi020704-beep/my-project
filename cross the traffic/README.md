# 🚗 CROSS THE ROAD

A fun arcade-style crossing game built with **Python** and the **Turtle graphics** library.

The objective is simple: help the player cross the road while avoiding moving cars. Every successful crossing increases the game's difficulty by making the cars move faster.

## 🎮 Gameplay

* Use the **Up Arrow (↑)** to move forward.
* Use the **Down Arrow (↓)** to move backward.
* Avoid all moving cars.
* Reach the top of the screen to complete a level.
* Each completed level increases the speed of the cars.
* The game ends when the player collides with a car.

## ✨ Features

* Randomly generated moving cars
* Collision detection
* Level progression
* Increasing difficulty
* Scoreboard displaying the current level
* Game Over screen
