# ✊📄✂️ Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game built with Python.

The player competes against the computer by choosing Rock, Paper, or Scissors. The computer randomly selects its move, and the game determines the winner based on the classic rules.

---

## 🎮 Features

* Play against the computer
* Random computer choices using Python's `random` module
* ASCII art representation for Rock, Paper, and Scissors
* Input validation for invalid choices
* Instant win/lose/draw results

---

## 📖 How to Play

Choose one of the following options:

| Number | Choice   |
| ------ | -------- |
| 0      | Rock     |
| 1      | Paper    |
| 2      | Scissors |

The computer will randomly choose its move, and the winner will be displayed.

### Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a Draw

---

## 🚀 Getting Started

### Prerequisites

* Python 3.x

### Run the Game

```bash
python rock_paper_scissors.py
```
